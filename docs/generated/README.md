# Generated References

These source-linked references are deterministic validation artifacts. Do not
edit them by hand.

- [Python symbols](python-symbols.md) covers every class, function, nested
  function, test, and method under `scripts/` and `tests/`.

Run `npm run docs:write` after changing declarations or docstrings. CI runs
`npm run docs:check` and fails when a declaration is undocumented or the
tracked reference is stale.
